# 🛒 Backend de E-commerce con NestJS

Este es el backend para una aplicación de e-commerce, diseñado para gestionar productos, usuarios y pedidos. Este proyecto está construido con NestJS, un framework de Node.js, y utiliza sqlite como base de datos.

## 🌟 Características

- 📦 Gestión de productos (CRUD).
- 🔒 Autenticación de usuarios.
- 🛒 Gestión de pedidos.
- 💳 Soporte para múltiples métodos de pago.
- 🚚 Integración con servicios de envío.

## 🛠️ Tecnologías Utilizadas

- NestJS
- MongoDB
- Mongoose
- JWT (JSON Web Tokens)
- bcryptjs
